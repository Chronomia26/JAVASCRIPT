What is JavaScript?

Imagine web pages are like clothing that you wear. You can only do so much with the clothes - they cover you and provide protection, but are mostly static. JavaScript is like adding buttons and zippers to the clothes. Instead of just passively wearing them, you can now interact with them. You can open, zip, and button your clothing to dynamically change how you're protected and uncover different parts as needed.

Similarly, web pages originally were mostly static content, like HTML and CSS that structured and styled the page. But just like adding interactive elements like buttons and zippers to clothes, JavaScript enables you to add dynamic functionality and effects to web pages. You can respond to user actions, detect clicks, change content, validate input, animate objects, load new data—things that make web pages feel engaging and responsive.

Zippers and buttons are cheap and lightweight versus extremely fancy and complex mechanisms yet enable so much extra functionality. Similarly, JavaScript code is relatively simple, lightweight and easy to learn, but unlocks tremendous interactivity. Adding a bit of JavaScript code, even to static HTML/CSS sites, brings pages to life just like adding zippers and buttons give plain clothes customizable functionality.

JavaScript Datatypes

One of the most fundamental characteristics of a programming language is the set of data types it supports. These are the type of values that can be represented and manipulated in a programming language. JavaScript allows you to work with three primitive data types −

  • Numbers, eg. 123, 120.50 etc.
  • Strings of text e.g. "This text is a string" etc.
  • Boolean e.g. true or false.

Note − JavaScript does not make a distinction between integer values and floating-point values. All numbers in JavaScript are represented as floating-point values. JavaScript represents numbers using the 64-bit floating-point format defined by the IEEE 754 standard.

Arithmetic Operators

Let's dive into the world of arithmetic with our clever cat friends. Just like cats gracefully navigate different surfaces, JavaScript arithmetic operators perform various calculations on numbers.

Imagine our cat, Whiskers, calculating her treats with the following operators:

  • Addition (+): Whiskers loves treats, and if we add more treats, she gets even happier.
    Example: let totalTreats = treatsInBowl + treatsFromYou;

  • Subtraction (-): Sometimes Whiskers shares treats with her friends. Subtraction helps us know how many treats are left.
    Example: let treatsLeft = totalTreats - treatsGiven;

  • Multiplication (*): Whiskers is a clever cat, and multiplication helps us calculate the total happiness when treats are shared with multiple cats.
    Example: let totalHappiness = happinessPerCat * numberOfCats;

  • Division (/): Dividing treats ensures each cat gets a fair share. Whiskers loves equality.
    Example: let treatsPerCat = totalTreats / numberOfCats;

  • Modulus (%): Sometimes we have treats left, and Whiskers wants to know the remainder after sharing. Modulus helps with that.
    Example: let leftoverTreats = totalTreats % numberOfCats;

Javascript Functions

"Paw-some Tricks Unleashed" 🐾🎩

A function in JavaScript is like a set of instructions or a block of code that you can name and reuse whenever you need it. It helps in organizing code and making it more manageable.

In Cat Terms... Think of functions as your very own talented cats. You can give them names (function names) and teach them tricks (code). Whenever you call their names, they perform those tricks. Let's dive into the fun world of JavaScript functions with our cat friend, Mittens!


Defining a Function - Naming Mittens 🐾

Let's start by creating a function and giving it a cat-related name, say, Mittens:

function mittens() {
//This is where you put tricks to teach mittens
}

In Cat Terms... You're basically naming a trick for Mittens to learn.


Teaching Mittens a Trick - Asking for Food 🍲

Now, let's teach Mittens a trick - asking for food politely:

function askForFood() {
alert("Meow meow, may I plz haz food?");
}

askForFood(); // Mittens performs her trick: "Meow meow, may I plz haz food?"

In Cat Terms... Every time you say "askForFood," Mittens performs her adorable trick, asking for food.


Javascript Parameters - Purr-sonalized Tags 🐾

In JavaScript, a function parameter is like a placeholder or variable that you define when creating a function. These placeholders allow you to pass values into the function when you call it, making the function more versatile and adaptable. Parameters act as inputs that the function can use to perform different actions based on the provided values.


function greetCat(name) {
alert("Meow there! I'm " + name + "!");
}

greetCat("Whiskers"); // Output: Meow there! I'm Whiskers!
