JavaScript Programs

JavaScript Statements

JavaScript statements are the building blocks of a program and can include values, operators, expressions, keywords, and comments. These statements are executed sequentially, one after the other. JavaScript Variables üêæ

Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword as follows.


Automatically Declared Variables

Automatically declared variables are created when you assign a value to an undeclared variable. JavaScript automatically understands the variable and assigns the value to it.

// Variable x is automatically declared
x = 5;

// Now x holds the value 5
console.log(x);


// Variable message is automatically declared
message = "Hello, World!";

// Now message holds the string "Hello, World!"
console.log(message);


// Variable count is automatically declared
count = 10;

// Now count holds the value 10
console.log(count);


Variables Declared Using var

In JavaScript, var was traditionally used to declare variables. Variables declared with var are function-scoped.

// Using var to declare a variable
var a = 2;

// Now a holds the value 2
console.log(a);

// Variables declared using var are hoisted
console.log(b); // Outputs: undefined
var b = 7;

// var allows redeclaration of variables
var name = "John";
var name = "Doe"; // No error, but the variable is overwritten
console.log(name); // Outputs: Doe


Variables Declared Using let

let was introduced in ECMAScript 6 (ES6) and provides block-scoping, which can be more predictable than var.

// Using let to declare a variable
let age = 25;

// Now age holds the value 25
console.log(age);

// Variables declared using let have block scope
if (true) {
let localVar = "I am inside a block";
}

console.log(localVar); // Error: localVar is not defined

// let does not allow redeclaration of the same variable in the same scope
let color = "red";
// let color = "blue"; // Error: Identifier 'color' has already been declared


Variables Declared Using const:

const is used to declare variables that should not be reassigned. It also has block-scoping.

// Using const to declare a constant variable
const PI = 3.14;

// Now PI holds the value 3.14
console.log(PI);

// Variables declared using const must be assigned a value during declaration
const gravity; // Error: Missing initializer in const declaration

// const variables cannot be reassigned
const daysInWeek = 7;
// daysInWeek = 8; // Error: Assignment to constant variable.
